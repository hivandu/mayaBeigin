<h5>本节概述</h5> <p>使用图表编辑器可以用图表的方式操纵动画曲线。动画曲线上的点表示关键帧；关键帧之间的跨度是曲线段；切线描述了曲线段进入和退出关键帧的方式。选择Window &gt; Animation Editors &gt;Graph Editor，打开图表编辑器。如图1.6.1所示。</p> <p><a target="_blank" title="image001" href="http://www.flickr.com/photos/95019520@N00/2346649975/"><img alt="image001" src="http://farm3.static.flickr.com/2347/2346649975_b843df527d.jpg" border="0" /></a>图1.6.1</p> <p>图表编辑器由下列部分组成：</p> <p>l 菜单栏</p> <p>l 工具栏</p>  <p>l 大纲列表</p> <p>l 图表视图</p> <p>图表编辑器仅使用关键帧和动画曲线工作，所以某些类型的曲线在此处是不可见的。例如使用表达式和反向动力学制作的动画，可以在图表编辑器中显示但是不能直接在图表编辑器中编辑。</p> <h5>使用图表编辑器菜单栏</h5> <p>Graph Editor 菜单栏包含着操纵图表区动画线和关键帧的各种工具和功能。</p> <h6>Edit（编辑）菜单</h6> <p><strong>Undo/Redo </strong><strong>命令</strong></p> <p>Undo 命令可取消一个工具或操作的作用，而Redo 与之相反。</p>  <p><strong>Cut </strong><strong>命令</strong></p> <p>此命令从选择的属性上去除现存的关键帧，并将一份信息拷贝放在关键帧剪贴板上。如果在Graph Editor 的图表区选择了一曲线段，当选择了Edit &gt; Cut 命令后，这个区域将被去除。</p> <p>如果没有选择曲线段，Cut 命令将作用于在Graph Editor 的提纲列表中的当前选项。</p> <p><strong>Copy </strong><strong>命令</strong></p> <p>Copy 命令从选择的属性上复制现存的关键帧，并将复制的信息放在关键帧剪贴板上。如果在图表区选择了曲线段，当用户选择Edit &gt; Copy 命令时，这个区域将被复制；如果没有选择曲线段，Copy 命令将作用于在Graph Editor 提纲列表中的当前选项。</p> <p><strong>Paste </strong><strong>命令</strong></p>  <p>此命令把关键帧剪贴板上的关键帧添加到某属性中。粘贴的关键帧可以并入现存的关键帧、在某个时间点插入关键帧（替换现存的关键帧），或在某个时间点重新放置关键帧。</p> <p><strong>Delete </strong><strong>命令</strong></p> <p>此命令从选择的属性上去除现存的关键帧。</p> <p><strong>Scale </strong><strong>命令</strong></p> <p>此命令可把某一范围内的关键帧扩展或压缩到一新的时间范围中。</p> <p>注意：如果在Graph Editor 中的提纲列表选择了属性，但是没有选择的曲线段，则缩放将作用于该属性的整个关键帧范围。</p> <p>当打开Only Scale Specified Key 选项时，可只缩放选择区域中的关键帧，而在选择区域外的关键帧不受影响。</p> <p><strong>Snap</strong><strong>（捕捉）命令</strong></p>  <p>使用捕捉命令，迫使选择的关键帧以整数时间单位（或整数属性值）移动。使用捕捉下面的Select Unsnapped 命令，可以选择不在整数单位的关键帧。在系统的默认设置下，将捕捉到最近的时间单位；可以选择Edit &gt; Keys &gt; Snap Keys - ，观察选项对话框，编辑各项设置。</p> <p><strong>Select Unsnapped </strong><strong>命令</strong></p> <p>选用此命令，将选择当前不处于整数时间单位的所有关键帧。</p> <h6>View（视图）菜单</h6> <p>此菜单中的命令控制图表编辑器“图表区”中可显示出的内容，从而设置动画曲线的哪些元素是可编辑的。</p> <p>当调整要观察的帧范围时，在Graph Editor 中，可以在图表区的水平方向和垂直方向缩放要观察的范围。</p>  <p><strong>Frame All </strong><strong>命令</strong></p> <p>此命令使所有动画曲线都显示出来。</p> <p><strong>Frame Selected </strong><strong>命令</strong></p> <p>此命令使选择的动画曲线显示在整个图表区中。</p> <p><strong>Frame Playback Range </strong><strong>命令</strong></p> <p>此命令调节图表区使其可显示出整个“播放范围”的动画信息。</p> <p><strong>Center Current Time </strong><strong>命令</strong></p>  <p>选择此命令，将改变视图显示，把当前时间移动到编辑器的中间位置。</p> <p><strong>Auto Frame </strong><strong>命令</strong></p> <p>选择此命令，可自动调节视图区的大小使其适合于选择动画曲线的显示。</p> <p><strong>Show Results </strong><strong>命令</strong></p> <p>如果不创建动画曲线类型的动画，例如路径动画和表达式，是整个动画的一部分，使用此命令可为这些类型的动画创建一个图形的表达。如果要编辑这些类型的曲线，通过将执行Bake Channel，将它们转换为动画曲线。</p> <p><strong>Show Buffer Curves </strong><strong>命令</strong></p> <p>选择此项，图表区将显示被编辑曲线的原始形状。</p>  <p><strong>Keys </strong><strong>命令</strong></p> <p>为了操作动画曲线的某个元素，首先需要使其可见，这时可以把它调入到图表编辑器中。默认的，在建模视图中选中的物体会被自动调入到图表编辑器中。</p> <p>在动画曲线中，关键帧由一个“黑点”来代表，当其被选中时，关键帧变为黄色高亮显示。</p> <p><strong>Tangents </strong><strong>命令</strong></p> <p>与关键帧相同，只有当切线元素是可视的，它才能被选择，从而才能被编辑。</p> <p>在图表区中切线是由褐色的切线手柄来代表的，并且与动画曲线的曲线段相切，当被选中时，变为高亮的绿色。</p> <p>注意:如果Unify Tangents 命令作用于曲线段，当用户编辑选择的手柄时，在关键帧两侧将统一地调整曲线。如果Break Tangent 在起作用，关键帧两侧的切线手柄将独立地作用于曲线。</p> <p><strong>Infinity </strong><strong>命令</strong></p>  <p>此命令设置动画曲线第一个关键帧之前和最后一个关键帧之后的曲线是否显示。</p> <p>注意:在系统的默认设置下，第一个关键帧之前和最后关键帧之后的切线将是连续的。</p> <h6>Select（选择）菜单</h6> <p>此菜单中的命令设置动画曲线中的那些元素是可被选择和可被编辑的。</p> <p><strong>All </strong><strong>命令</strong></p> <p>此命令设置动画曲线的所有元素都是可选择的。</p> <p><strong>Curve </strong><strong>命令</strong></p> <p>此命令设置只有曲线段才能被选择。</p>  <p><strong>Key </strong><strong>命令</strong></p> <p>此命令设置只有关键帧才能被选择。</p> <p><strong>In Tangent </strong><strong>命令</strong></p> <p>此命令设置只有关键帧的In Tangent（入切线手柄）才能被选择。</p> <p><strong>Out Tangent </strong><strong>命令</strong></p> <p>此命令设置只有关键帧的Out Tangent（出切线手柄）才能被选择。</p> <h6>Curves（曲线） 菜单</h6>  <p>Curves 菜单中的各种功能项目用于处理整个动画曲线。</p> <p><strong>Pre Infinity </strong><strong>和</strong><strong>Post Infinity </strong><strong>动画曲线选项</strong></p> <p>Pre Infinity 和Post Infinity 分别定义了第一个关键帧以前和最后一个关键帧之后的动画曲线行为。Pre infinity 项包括5 种类型的设置。</p> <p>l <strong>Cycle</strong>（循环） 选择此命令，将无限地重复动画曲线的形状，如图所示。</p> <p>l <strong>Cycle with Offset</strong>（偏移循环） 选择此命令，将无限地重复动画曲线的形状，而且它把循环曲线最后一个关键帧的数值添加到原始曲线中第一个关键帧的数值上。</p>  <p>l <strong>Oscillate</strong>（振荡） 此项通过反转动画曲线数值的方式来重复动画曲线，因此在曲线形状上，产生来回震荡的效果。</p>
<p>l <strong>Linear</strong>（线性） 此项设置利用曲线两端关键帧的切线信息外推其值，产生延伸至无穷远处的线性曲线。如图所示。</p> <p>l <strong>Constant</strong>（恒量） 此项设置将保持曲线第一个或最后一个关键帧的数值，这是Maya 系统的默认设置。</p> <p><strong>Curve Smoothness </strong><strong>命令</strong></p> <p>此命令设置动画曲线显示的是粗糙或是平滑，但并不影响曲线实际的作用。</p>  <p>此命令包含有子菜单，分别控制显示的平滑程度。</p> <p><strong>Bake Channel</strong><strong>（仿真通道）命令</strong></p> <p>对于某个属性，此命令从所有起作用输入节点中选择一个节点，并根据此节点为此属性重新计算出一个新的动画曲线。</p> <p>注意：这个功能和Edit &gt; Keys &gt; Bake Simulation 功能有所区别。</p> <p><strong>Simplify Curve</strong><strong>（简化曲线）命令</strong></p> <p>此命令去除对动画曲线的形状不起作用的关键帧。当手工添加关键帧或执行像Bake Channel 这类命令时，将产生过剩的关键帧。用户可能需要去除动画曲线上的关键帧，减少图表区的复杂性，并且加大曲线跨度，调整曲线切线。</p>  <p><strong>Spreadsheet </strong><strong>命令</strong></p> <p>此命令可把选择动画曲线的数值和属性显示在属性编辑器中，并且可进行观察和编辑。</p> <p><strong>Buffer Curves</strong><strong>（缓冲曲线）</strong></p> <p>使用Buffer Curve Snapshot 和Swap Buffer Curves 可比较当前动画曲线和早先动画曲线的形状。</p> <p>选择View &gt; Show Buffer Curves 命令，打开缓冲曲线显示。选择动画曲线，使用Curves &gt; Buffer Curve Snapshot 命令，或者按下图表编辑器的Buffer Curve Snapshot 按钮，快照曲线。编辑动画曲线。此时可看到代表原始曲线的一条灰色曲线。</p> <p>使用Curves &gt; Swap Buffer Curve 命令或按下图表编辑器的Swap Buffer Curve 按钮，将缓冲曲线和已编辑曲线交换。通过来回交换这些曲线来测试动画。</p>  <p><strong>Weighted Tangents</strong><strong>（权重切线和非权重切线）</strong></p> <p>系统默认设置为非权重切线，它提供了操纵切线角度的简单手柄。为改变动画曲线的权重，使用Curves &gt; Weighted 去改变曲线的切线类型以设置权重。</p> <p>用户不能在同条曲线上混合权重和非权重切线。权重切线以图表的形式表明了切线的长度对动画曲线段地影响，当Free Tangent Weight 处于有效状态时，可以同时快速直观地编辑切线角度和影响。</p> <h6>Keys（关键帧）菜单</h6> <p><strong>Break Tangents</strong><strong>（断开切线手柄）命令</strong></p> <p>选用此命令，则用户可单独操作一个关键帧的入切线手柄和出切线手柄，而无需担心它们相互影响。</p> <p><strong>Unify Tangents</strong><strong>（统一切线手柄）</strong></p>  <p>当选择此命令时，则一个关键帧的“入切线手柄”和“出切线手柄”相当于一个切线手柄，调节其中的任何一个手柄都会影响另一个手柄相同的数量（选择Keys&gt;BreakTangents 命令后单独编辑切线）。</p> <p><strong>Lock Tangent Weight</strong><strong>（锁定切线权重）命令</strong></p> <p>当选用此命令时，可锁住切线的权重。例如当移动切线时，只有切线的角度的改变，权重保持不变。</p> <p><strong>Free Tangent Weight(</strong><strong>释放切线权重)</strong></p> <p>此命令设置当编辑切线手柄时，它的角度和权重都是可调节的。</p> <p><strong>Convert to Key </strong><strong>命令</strong></p> <p>转化选择的关键帧为Breakdowns。</p>  <p><strong>Convert to Breakdown </strong><strong>命令</strong></p> <p>转化选择Breakdowns 为正常的关键帧。</p> <p><strong>Add Inbetween </strong><strong>命令</strong></p> <p>在当前时间插入Inbetween。</p> <p><strong>Remove Inbetween </strong><strong>命令</strong></p> <p>在当前时间去除一个Inbetween。</p> <h6>Tangents(切线)菜单</h6>  <p>此菜单描述了曲线段从关键帧的进入和退出。此菜单操纵选择关键帧左右曲线段的形状。这些设置仅影响现存动画曲线段的切线。</p> <p><strong>Spline</strong><strong>（样条曲线）命令</strong></p> <p>此命令在选择的关键帧之前的关键帧和之后的关键帧之间创建比较平滑的动画曲线。曲线的切线是共线的（都具有相同的角度）。这确保动画曲线平滑的进入和退出关键帧。用户可以通过单击工具栏上的图标来选择此种类型的切线。当需要创建流畅的运动效果时，样条切线方式是极好的选择。用户可以使用最小数目的关键帧来达到需要的效果。</p> <p><strong>Linear</strong><strong>（线性）命令</strong></p> <p>使用此命令，使创建的动画曲线为直线连接两个关键帧。</p> <p>如果入切线的类型为线性，则关键帧前面的曲线段是直线。如果出切线的类型为线性，则关键帧之后的曲线段是直线。用户可以通过单击工具栏上的图标来选择此种类型的切线。</p> <p><strong>Clamped</strong><strong>（夹具）命令</strong></p>  <p>此命令创建的动画曲线可既有样条曲线的特征又有直线的特性。</p> <p>当两个临近关键帧的属性数值非常接近时，两个关键帧之间的动画曲线为直线；否则，两个临近关键帧之间的动画曲线为样条曲线。</p> <p><strong>Stepped</strong><strong>（台阶）命令</strong></p> <p>此命令创建台阶状的动画曲线，其关键帧的“出切线手柄”是直线。因为曲线段是平直的（水平的），使一个关键帧到另一个关键帧水平方向的属性数值没有变化。适合制作灯的开关动画。</p> <p><strong>Flat</strong><strong>（平直）命令</strong></p> <p>此命令使关键帧的入切线和出切线为水平的，即向量的斜率为零度。用户也可选择工具栏上的图标来选择此命令。当球体升至高空时，用户将注意到球体在空中停留一短暂的时间。通过使用平直切线，用户可以创建这个效果。</p> <p><strong>Fixed</strong><strong>（固定）命令</strong></p>  <p>使用这种类型的切线，当编辑关键帧时，关键帧切线保持不变。当制作球体动画时，用户将发现，球体不是在所希望的高度下落，但是地板和其它事物的影响是完美的。可采用“fixed”切线类型，只改变球体的高度，但是保持切线的角度。</p> <p><strong>In Tangents</strong><strong>（命令）和Out Tangents 命令</strong></p> <p>用户可以为进入关键帧和退出关键帧的切线指定不同的类型。</p> <h5>使用图表编辑器工具栏</h5> <h6>移动最近关键帧工具（Move Nearest Picked Key Tool）</h6> <p><a target="_blank" title="image003" href="http://www.flickr.com/photos/95019520@N00/2346649921/"><img alt="image003" src="http://farm3.static.flickr.com/2378/2346649921_e50126f3e4.jpg" border="0" /></a>此工具可让用户快速地使用鼠标移动单独的关键帧或切线手柄。</p> <p>此工具和移动工具不同。它可操作激活曲线上最近的关键帧或切线手柄，而不必精确地选择它们。</p> <h6>插入关键帧工具（Insert Keys）</h6>  <p><strong><u><font color="#cc0000"><a target="_blank" title="image005" href="http://www.flickr.com/photos/95019520@N00/2346649885/"><img alt="image005" src="http://farm3.static.flickr.com/3077/2346649885_1996ae26e7.jpg" border="0" /></a></font></u></strong>使用此工具可在选择的动画曲线上插入新的关键帧，使用左键选择要插入关键帧的曲线。</p> <p>按住鼠标中键沿动画曲线拖动，在要插入关键帧的位置释放中键，则创建了一个新的关键帧。新关键帧的切线将保持原动画曲线段的形状。</p> <h6>添加关键帧工具（Add Keys）</h6> <p><a target="_blank" title="image00<br>
7" href="http://www.flickr.com/photos/95019520@N00/2346649829/"><img alt="image007" src="http://farm3.static.flickr.com/2035/2346649829_2a9b342ea6.jpg" border="0" /></a>此工具可在选择动画曲线的任意位置处添加关键帧。</p> <p>单击鼠标中键决定新关键帧的位置，释放鼠标中键放置关键帧。新关键帧的切线与临近关键帧的切线类型是相同的。</p> <h6>晶格变形关键帧工具(Lattice Deform Keys)</h6> <p><a target="_blank" title="image009" <h6>关键帧状态栏（Key Stats fields）</h6> <p><a target="_blank" title="image011" href="http://www.flickr.com/photos/95019520@N00/2346649747/"><img alt="image011" <p>图表编辑器的图表区显示了动画曲线的曲线段、关键帧和关键帧切线。在图表区的任意位置右击可出现一弹出菜单，包含有图表编辑器的菜单项目。使用图表编辑器的菜单命令或工具栏中的工具可编辑和调节动画曲线。也可以使用选择工具、移动工具和缩放工具去操纵Graph Editor 中图表区的关键帧。</p> <h6>选择工具</h6> <p>使用选择工具可以选择曲线、曲线段、切线和在图表区的关键帧。</p> <p>使用鼠标拖拽选择一个区域的项目，或者在动画曲线上、切线手柄或关键帧单击，选择单个项目。</p> <h6>移动工具</h6> <p>使用移动工具可以选择和操纵视图区的成分。</p> <p>选择要操纵的成分，然后选择移动工具（就象在选择工具中描述的那样）。使用鼠标中键移动关键帧点、切线手柄和曲线。</p>  <h6>缩放工具</h6> <p>使用缩放工具可以缩放动画曲线段的范围和关键帧的位置。可以使用两种方式缩放视图区的关键帧，用户可以在缩放工具设置对话框选择缩放方式。</p> <p>如果要访问缩放工具设置对话框，当Graph Editor 是当前激活对话框时，双击缩放工具图标，打开工具设置对话框。</p> <h6>缩放工具设置</h6> <p>Manipulator——选择这个选项，用户可以使用操纵器盒的方式缩放关键帧。</p> <p>Gestural——当选择这种缩放方式，当缩放关键帧时，将以鼠标的位置确定缩放向量。</p> <h6>使用操纵器(Manipulator)缩放方式缩放</h6> <p>1 当使用操纵器缩放动画范围时，在缩放工具设置对话框将缩放方式设置为Manipulator。</p> <p>2 关闭工具设置对话框，使用鼠标左键选择缩放范围。则一个白色操纵器方形盒出现在视图区，在操纵器的每个角上有一个手柄，方形盒包围着选择的区域。</p>  <p>3 使用鼠标左键在操纵器的<br />
角上拖拽，可以拉伸和压缩选择的区域。使用鼠标中键可以重新放置选择的区域。当使用操纵器时，临近缩放区域的曲线段也被改变。如图1.6.3所示。</p> <p><strong><u><font color="#cc0000"><a target="_blank" title="image039" href="http://www.flickr.com/photos/95019520@N00/2347480700/"><img alt="image039" src="http://farm3.static.flickr.com/3262/2347480700_402d2d5b2a.jpg" border="0" /></a></font></u></strong>图1.6.3</p> <h6>使用Gestural 缩放方式进行缩放</h6> <p>1 如果要使用Gestural 缩放方式进行缩放，在缩放工具设置对话框将缩放方式设置为Gestural。</p> <p>2 关闭工具设置对话框，使用鼠标左键选择缩放的区域。</p> <p>3 使用鼠标中键单击，选择缩放轴点，水平拖拽鼠标，缩放选择区域的时间范围；垂直拖拽鼠标，可以缩放选择区域的数值信息。</p> <p>水平和垂直的两条实体线相交于一个方形盒，表明了缩放操作的轴点。如图1.6.4所示。</p> <p><a target="_blank" title="image041" href="http://www.flickr.com/photos/95019520@N00/2346650093/"><img alt="image041" src="http://farm3.static.flickr.com/2325/2346650093_a96d5de72b.jpg" border="0" /></a>图1.6.4</p></p>
